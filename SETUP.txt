==========================================================
   TIFF'S PINK ROOM  --  HOW TO START IT
==========================================================

  >>>  Double-click:   START HERE.bat   <<<

  That's basically it. The first time, it sets EVERYTHING up
  for you automatically:
     - checks your PC and picks the right AI model for it
     - installs Python (if you don't have it)
     - installs LM Studio, the AI engine (if you don't have it)
     - downloads the AI model (a few GB - this is the slow part)
     - opens Tiff's Pink Room in your web browser

  While it works:
     - If Windows pops up asking "Do you want to allow this?"
       -> click YES.
     - The first run takes a while (it's downloading a few GB).
       A black window shows what it's doing. Let it cook.
     - When your browser opens to the Pink Room, you're in.

  Every time after the first:  just double-click START HERE
  again and it opens in a few seconds.

  To STOP it:  close the black window.


----------------------------------------------------------
 IF SOMETHING GETS STUCK
----------------------------------------------------------
  - Browser didn't open?  Wait a bit, then type this into your
    web browser:   http://localhost:7777

  - It said it couldn't install Python or LM Studio?  Your
    Windows might be older. Install them by hand:
       Python:    https://www.python.org/downloads/
                  (CHECK the box "Add python.exe to PATH"!)
       LM Studio: https://lmstudio.ai
    Then run START HERE again - it'll finish the rest.

  - Tiff won't answer?  Open LM Studio, click the "Developer"
    (or "Local Server") tab on the left, and press "Start
    Server" if it isn't green. Then refresh your browser.

  - Still stuck?  Text Bryan. He's got you.
